#### Dictionaries
Accessing and changing dictionaries
```python
dict[key] # index value of key
dict[key] = value # adds entry or modifies if it already exists
del dict[key] # deletes key
key in dict # tests to see if key exists
```

Dictionary Syntax
```python
dict = {
		'John': '1',
		'Alan': 67,
		30: 5.0
}
```
