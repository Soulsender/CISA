#### IPv4 Packet Header
- 14 fields, 13 required
- headers in order
	- version (4)
	- fragment offset
	- time to live
	- protocol
	- header checksum 
	- source address
	- destination address

#### IPv6 Packet Header
- headers in order
	- version (6)
	- flow label
	- payload length
	- hop limit
	- source address
	- destination address
